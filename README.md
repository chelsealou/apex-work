# apex-work

A collection of Salesforce Apex triggers and a Python project workspace.

## Structure

- `apex/` — Salesforce Apex triggers (AccountAddress, ClosedOpportunity, CreateUser)
- `src/` — Python source code
- `requirements.txt` — Python dependencies

## Getting Started

```bash
# Create a virtual environment
python3 -m venv .venv
source .venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Run
python src/main.py
```
